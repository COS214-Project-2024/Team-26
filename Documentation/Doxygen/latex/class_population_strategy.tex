\doxysubsection{Detailed Description}
Abstract base class defining a strategy for population-\/related operations in the city simulation. 

The \doxylink{class_population_strategy}{Population\+Strategy} class provides a base interface for strategies that affect the city\textquotesingle{}s population. Derived classes implement strategies by overriding the \doxylink{class_population_strategy_ae2fdfc4da2dfa60d524459d66c48ba17}{execute()} method. This class also provides helper methods to access relevant population statistics through the \doxylink{class_stats_mediator}{Stats\+Mediator}.
