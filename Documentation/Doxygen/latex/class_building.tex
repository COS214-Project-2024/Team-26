\doxysubsection{Detailed Description}
Abstract base class representing a building in the simulation. 

The \doxylink{class_building}{Building} class implements the State pattern, where different states (Placed, Under\+Construction, Complete, Demolished) affect the building\textquotesingle{}s behavior. Each building has a location, space capacity, and various consumption metrics.
