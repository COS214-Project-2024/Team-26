\doxysubsection{Detailed Description}
Abstract base class for the \doxylink{class_building}{Building} State Pattern. 

\doxylink{class_building_state}{Building\+State} defines the interface for all concrete building states (Placed, Under\+Construction, Complete, Demolished). Each state implements different behavior for resource consumption and building metrics based on the current phase of the building\textquotesingle{}s lifecycle.
