\doxysubsection{Detailed Description}
Represents a building that has been demolished. 

The \doxylink{class_demolished_state}{Demolished\+State} class implements the \doxylink{class_building_state}{Building\+State} interface, defining the characteristics of a building when it is in a demolished state. All costs, consumption, and production values are set to zero, as the building is not operational.
