\doxysubsection{Detailed Description}
A class representing landmark buildings in the city, such as monuments and parks. 

\doxylink{class_landmark_building}{Landmark\+Building} is derived from the \doxylink{class_building}{Building} base class and implements functionality specific to landmark structures that contribute to city aesthetics and tourism.
