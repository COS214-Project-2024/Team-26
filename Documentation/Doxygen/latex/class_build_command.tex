\doxysubsection{Detailed Description}
Represents a command to construct a building at a specific location. 

The \doxylink{class_build_command}{Build\+Command} class specifies the type of building to be constructed and the location (coordinates) where it should be built. It uses a mediator to communicate and execution of the building process.
