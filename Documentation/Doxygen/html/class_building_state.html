Detailed Description

Abstract base class for the Building State Pattern.

BuildingState defines the interface for all concrete building states (Placed, UnderConstruction, Complete, Demolished). Each state implements different behavior for resource consumption and building metrics based on the current phase of the building's lifecycle.
